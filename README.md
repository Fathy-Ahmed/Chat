# Real-time chat feature using ASP.NET Core SignalR that enables two or more users to exchange messages instantly.

<h3>Functional Requirements</h3>

  
<h4>1. User Connection</h4>
  <ul>
    <li>Connect to the SignalR Hub after login.</li>
    <li>Display the user's connection status. </li>
    <li>Handle automatic reconnection.</li>
    <li>Track online/offline users.</li>
  </ul>


<h4>2. One-to-One Chat</h4>
   <ul >
    <li>View a list of available users.</li>
    <li>Select a user.</li>
    <li>Send messages.</li>
    <li>Receive messages instantly.</li>
    <li>View chat history.</li>
  </ul>
